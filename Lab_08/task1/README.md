В данной работе реализован паттерн для оптимитизации памяти при создании 10к деревьев 

Общие данные включают в себе тип дерева, цвет, размер и вид

Уникальными являются позиция, масштаб, вращение 

Логика: TreeFactory хранит уникальные типы деревьев. ForestRenderer хранит только координаты

В результате создаются 10000 деревьев а используется всего 4 flyweight обьекта 

Экономия памяти примерно 80-90 процентов
