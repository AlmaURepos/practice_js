# Документация паттерна: Безопасный парсинг данных

**Описание:**
Паттерн позволяет обрабатывать данные без риска аварийного завершения приложения.

**Контекст:**
Используется в системах, которые получают данные из ненадежных (чаще всего внешних) источников.

**Постановка проблемы:**
Дублирование вызова JSON.parse приводит к усложнению поддержки кода, когда одна и та же функция повторяется в разных местах.

**Решение:**
Принять входные данные, выполнить попытку парсинга, перехватить возможные ошибки и вернуть корректное значение.

**Дизайн:**
Вся логика обработки ошибок инкапсулируется внутри одной утилитарной функции.

**Реализация:**
Реализуется с использованием try-catch и параметра callback.

**Пример:**
См. файл pattern_example.js

**Последствия:**
- Повышение стабильности приложения
- Снижение дублирования кода
