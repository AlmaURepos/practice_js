### Что решает Factory?

данный паттерн предоставляет интерфейс для создания обьектов при этом конкретный класс опредлеяется во время выполнения 

### Отличие от конструктура 

пользвоатель не знает конкретный класс 
пример:

```javascript
new EmailNotification(...) //constructor

NotificationFactory.create('email', options ) //factory

```

### Когда использовать?

- тип обьекта определяется во время выполнения кода 
- требцется полиморфизм
