Паттерн Singleton обеспечивает чтобы у класса только один экземпляр и предоставляет глобальную точку доступа к нему

- Когда использовать?

логгер, кэш, подключение к бд

- Преимущества

контроль создание обьекта, глобальный доступ и экономия ресурсов 

- Недостатки

сложность тестирования и может нарушить принцип единственный ответственности
